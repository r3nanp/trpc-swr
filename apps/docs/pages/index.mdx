# tRPC-SWR

![image](https://raw.githubusercontent.com/sachinraja/trpc-swr/main/assets/social.png)

[tRPC](https://trpc.io)-ified [SWR](https://swr.vercel.app) hooks.

## What is tRPC-SWR

tRPC-SWR is a set of hooks that allow you to use SWR with tRPC. It hooks directly into tRPC's native client and provides a set of SWR hooks.
